# Planify Server

This is the backend part for the Planify application.

### Project Structure

- `src/` - Source code for the application
- `prisma/` - Prisma schema and migration files
- `dist/` - Compiled JavaScript files

### Technologies Used

- Node.js
- Express
- Prisma
- SQLite
- TypeScript
